Wireframe Sphere

The sphere is generated using a subdivision method.

Input:    4 points on the unit sphere and these points are connected to form a tetrahedron.
The user interface allows the user to specify the number of subdivisions.
Output:    Triangulated sphere displayed in wireframe mode using LINE_LOOP for each triangle.
